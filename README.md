# Durão Despachante - Landing Page

Landing page para o serviço de despachante Durão Despachante.

## Deploy no GitHub Pages

Para hospedar este site gratuitamente no GitHub Pages:

1. Crie um novo repositório no GitHub (ex: `durao-despachante-landing`)
2. No terminal, execute:
   ```
   git remote add origin https://github.com/SEU_USERNAME/durao-despachante-landing.git
   git branch -M main
   git push -u origin main
   ```
3. No GitHub, vá para Settings > Pages
4. Selecione "Deploy from a branch" e escolha `main` como branch
5. Clique em Save
6. O site ficará disponível em: `https://SEU_USERNAME.github.io/durao-despachante-landing/`

## Arquivos

- `index.html`: Página principal
- `styles.css`: Estilos CSS
- `assets/logomarca.png`: Logo da empresa
